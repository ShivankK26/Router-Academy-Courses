## Basic Installations

- Install ReactJS Application by running the below Command - 

    ```
    npm create vite@latest
    ```

- Install some Packages -

    ```
    npm install ethers@5.7.2 axios
    ```

- Run the Server by the Below Command -

    ```
    npm run dev
    ```
