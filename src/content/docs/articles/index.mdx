We Welcome Contributions from the Community on Router Protocol Ecosystem. Here's how you can Contribute an Article & Earn Rewards -

## Guidelines for Writing Articles

1. Choose a Topic related to Router Protocol which you feel can add Value to the Community.
2. Write Clear, Concise, and well-structured Content.
3. Include Relevant Examples and Code Snippets, where appropriate.
4. Proofread your Article for Grammar and Technical Accuracy.

## How to Submit an Article

1. Fork the Router Academy Repository on [GitHub](https://github.com/ShivankK26/Router-Academy-Courses).
2. Create a new Markdown file in the `src/content/docs/articles/` Directory in the format `your-article-title.mdx`.
3. Write your Article using Markdown format.
4. Submit a Pull Request with your new Article.
